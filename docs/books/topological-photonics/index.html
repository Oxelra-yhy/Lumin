Topological Photonics from Maxwell's Equations

从 Maxwell 方程出发组织拓扑光子学的基础框架，覆盖光子能带、Berry 曲率、Chern 数、边界态和体边对应，并延伸到 Floquet、Weyl 与非线性主题。 A Maxwell-equation-first introduction to topological photonics, covering photonic bands, Berry curvature, Chern numbers, edge states, and bulk-boundary correspondence, with extensions to Floquet, Weyl, and nonlinear topics.
