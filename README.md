# Lumin

**面向人类研究人员和 Agent 的 open knowledge books。**

[在线站点](https://oxelra-yhy.github.io/Lumin/) · [Oxelra](https://oxelra.com/)

Lumin 用一个克制、可追踪的入口呈现 Oxelra 正在开放的知识书稿：包括公开预览页面、PDF 草稿、源稿链接以及面向章节、公式、图表、引用和可复现细节的 Issue 审议通道。

这个项目并不试图把草稿包装成最终出版物。它更接近一组开源、可审议、可持续修订的专注型 books：为人类研究人员提供清晰的阅读路径，也为 Agent 提供结构化、可引用、可复核的知识材料。

## 书稿

- [Topological Photonics from Maxwell's Equations](https://github.com/Oxelra-yhy/Topological-Photonics)  
  从 Maxwell 方程出发组织拓扑光子学的基础框架，把模式理论、周期介质、Bloch 能带和几何相位放在同一条叙事线上。书稿覆盖 Berry 曲率、Chern 数、边界态与体边对应，并进一步延伸至 Floquet 系统、Weyl 点、合成维度和非线性量子边界通道等前沿主题。

- [Non-Hermitian Photonics](https://github.com/Oxelra-yhy/Non-Hermitian)  
  以开放光学系统为主线，讨论损耗、增益、色散、辐射边界和散射极点如何改变传统本征模图景。书稿进一步展开准正规模、异常点、PT 对称、非布洛赫拓扑、皮肤效应、BIC 与器件物理边界等主题，强调模型假设、谱结构和实验可复现性之间的关系。

两部书稿目前均处于公开审议前草稿阶段。读者可以通过对应仓库的 Issues 提交概念跳跃、推导疑点、引用线索、图表问题和实验可复现性反馈。

## 阅读与审议

Lumin 收录的 open knowledge books 仍处在持续修订中，适合以“研究共同体预审”的方式阅读：先把它们视为可讨论、可追踪、可修正的工作稿，再围绕具体章节提出可定位的问题。

尤其欢迎以下类型的反馈：

- 章节结构是否清晰，概念过渡是否自然
- 公式推导、符号约定和物理假设是否一致
- 图表、数值流程和实验描述是否足够可复现
- 关键文献、历史脉络和术语边界是否需要补充

Lumin 的目标是保持入口轻量、来源明确、状态透明，让开放知识书稿从生成、复核到公开审议的每一步都能被人类研究人员和 Agent 看见。
